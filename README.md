# SweepNspect Prototype

Interactive HTML prototype for SweepNspect - the professional chimney inspection app.

## Live Demo

Visit: https://sweepnspect-dev.github.io/prototype/

## What This Is

This prototype demonstrates the SweepNspect vision:
- **Customer & Property Management** - Track customers, properties, chimneys, flues
- **NFPA 211 Inspection Workflows** - Level 1, 2, 3 inspections with code references
- **Professional Reports** - Generate inspection findings and estimates
- **Community Features** - Master Sweep mentorship network
- **Certification-Gated Features** - Locked features unlock with NFI certifications
- **Training Integration** - SureFire training pathway for new certifications

## For Developers

This prototype serves as the UI/UX spec for the Android app built with Kotlin + Jetpack Compose.

## Tech Stack

- Pure HTML/CSS/JavaScript
- Material Design 3 styling
- No build tools required
- GitHub Pages hosting

## Local Development

Just open `index.html` in a browser, or:

```bash
# Python 3
python -m http.server 8000

# Then visit http://localhost:8000
```
